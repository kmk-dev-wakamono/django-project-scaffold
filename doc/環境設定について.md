# 環境設定について

環境設定には Scoop を利用してください。

## Scoop について

Windows 向けのパッケージマネージャーです。  
管理者権限が不要かつ自動的に Path の設定等も行うため、セットアップの手間が少なくなります。
また、バージョンのアップグレードなどもコマンド1つで行えるため、ソフトウェア毎にバージョンアップをする必要がなくなります。

※パッケージマネージャは、iOS でいうところの App Store, Android でいうところの Play Store のような存在です。パッケージ（アプリケーション）を管理するやつです。

## Scoop のインストールについて

1. [Scoop の公式サイト](https://scoop.sh/#/) にアクセスします。
2. 自分のマシンで PowerShell を起動します。
3. PowerShell に **QuickStart** に記載してあるコマンドを貼り付け、実行します。（複数行貼り付けの際に警告が出るかもしれませんが、無視してOKです）
4. インストールが完了したら、インストールについては作業終了です。

## 利用するツールについて

最低限、以下のツールについて、使えることを確認してください。  
PowerShellであれば、 `Get-Command git` や `gcm git` で確認できます。  
※ツールがなければ、「用語 'git' は、コマンドレット、関数、スクリプト ファイル、または操作可能なプログラムの名前として認識されません。」というエラーが出てくると思います。

- git
- Python

上記ツールでエラーが出たら、以下コマンドを実行してください。

**git**  
`scoop install git`

**python 3.13**  
`scoop bucket add versions`  
`scoop install python313 #(3.13.12 がインストールされます。)`
